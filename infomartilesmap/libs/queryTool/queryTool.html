Display only:

This form can be used to display the sample points according to 2 different Folk Classification schemes where particle size analysis has been completed. The markers can be colour coded according to INFOMAR modified Folk Classification, the legend for this scheme is displayed in the Map Legend window below. They can also be displayed according to a modified Folk classification used by the British Geological Survey for their 1:250,000 scale seabed sediments map series and DigSBS250 digital dataset, further adjusted by Flemming as described in Geo-Seas publication Standards for seabed habitat mapping (Part A: Sediment).

Modified Folk - BGS Colour Code


This shows the colours applied to the markers of the map according to either the INFOMAR Modified Folk classification scheme or the British Geological Survey's (BGS) modified Folk classification scheme depending on which scheme is currently displayed on the map. The BGS Colour Code shows the colours applied to the markers on using a modified Folk classification used by the BGS for their 1:250,000 scale seabed sediments map series and DigSBS250 digital dataset, further adjusted by Flemming as described in GeoSeas publication Standards for seabed habitat mapping (Part A: Sediment).
